# JAVA FULLSTACK USER CRUD WEB APP

CRUD Application using React and Spring Boot

This project is a simple CRUD (Create, Read, Update, Delete) application built with React for the frontend and Spring Boot for the backend. It allows  for managing a collection of items.


=> Project Structure

-> The project is structured as follows:

client/: Contains the React frontend code.
server/: Contains the Spring Boot backend code.
database/: Contains database schema and migration scripts.

=> Technologies Used

* React
* Spring Boot
* Java
* H2 database
* npm
* Maven
